HFC INPUTS AUTO-POPULATOR — HOW TO RUN
========================================

GETTING THE APP (for colleagues)
---------------------------------
1. Install Git if you don't have it: https://git-scm.com/download/win
2. Install Python if you don't have it: https://www.python.org/downloads/
   (check "Add Python to PATH" during install)

3. Open Command Prompt and run:

   git clone https://github.com/YOUR_USERNAME/hfc-inputs-automation.git
   cd hfc-inputs-automation

4. Install dependencies:

   python -m pip install -r requirements.txt

5. Start the app:

   python -m streamlit run app.py

6. Your browser opens automatically at http://localhost:8501 — you're ready!


GETTING UPDATES (when the app is improved)
-------------------------------------------
Open Command Prompt in the app folder and run:

   git pull


TWO WAYS TO USE THE APP
-----------------------
Tab 1 - "Upload Files":
   Upload the XLSForm (.xlsx) and HFC template (.xlsm) directly from
   your computer. No credentials needed. Best for everyday use.

Tab 2 - "Google Drive & Box":
   Sign in with your Google work email to access survey forms that are
   shared with specific users. Paste the Google Drive link, and the app
   downloads and populates the file automatically.


SETTING UP GOOGLE SIGN-IN (admin only — one-time setup)
---------------------------------------------------------
To enable "Sign in with Google" for work-email-restricted files:

1. Go to: https://console.cloud.google.com/
2. Create a new project (or select your existing IPA project).
3. Go to "APIs & Services" > "OAuth consent screen":
   - User type: Internal (if using Google Workspace) or External
   - Fill in app name and contact email
4. Go to "APIs & Services" > "Credentials":
   - Click "Create Credentials" > "OAuth client ID"
   - Application type: Web application
   - Name: HFC Inputs App
   - Authorized redirect URIs — add BOTH:
       http://localhost:8501           (for local use)
       https://your-app.streamlit.app  (replace with your Streamlit Cloud URL)
   - Click Create, then copy the Client ID and Client Secret
5. Enable the Google Drive API:
   - Go to "APIs & Services" > "Library"
   - Search "Google Drive API" and click Enable

6. Add credentials to Streamlit Community Cloud:
   - Go to your app on share.streamlit.io > Settings > Secrets
   - Add:
       GOOGLE_CLIENT_ID = "your-client-id.apps.googleusercontent.com"
       GOOGLE_CLIENT_SECRET = "your-client-secret"
       GOOGLE_REDIRECT_URI = "https://your-app.streamlit.app"

7. For local use, create a file:
   .streamlit/secrets.toml
   and add the same three lines above, but with:
       GOOGLE_REDIRECT_URI = "http://localhost:8501"
